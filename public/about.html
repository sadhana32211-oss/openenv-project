🌍 About OpenEnv

A Standard Environment Interface for AI Agents

What is OpenEnv?

OpenEnv is a standardized environment interface for training and testing reinforcement learning (RL) agents. It provides a simple yet powerful API that follows the classic step() / reset() / state() pattern used in popular RL frameworks.

Current Environment: GridWorld-v0

The current implementation features a Grid World environment - a classic reinforcement learning problem where an agent navigates a 5×5 grid to reach a goal while avoiding obstacles and pits.

Environment Details:

  • Agent: Starts at position (0, 0)
  • Goal: Located at position (4, 4) - Reward: +10
  • Obstacles: Walls that block movement - Penalty: -0.5
  • Pits: Trap zones - Penalty: -1
  • Step Penalty: -0.01 per step (encourages efficiency)
  • Boundary: Hitting the edge - Penalty: -0.1
  • Max Steps: 100 (episode ends if not reached goal)

API Endpoints

POST /api/reset Reset environment to initial state
POST /api/step Execute action (body: { action: 0-3 })
GET /api/state Get current environment state
GET /api/env/info Get environment metadata
GET /api/envs List all active environments
DELETE /api/env/:env_id Delete an environment instance

Action Space

Action 0 ↑ Up
Action 1 → Right
Action 2 ↓ Down
Action 3 ← Left

Getting Started


// 1. Start the server
npm install
npm start

// 2. Reset the environment
curl -X POST http://localhost:3000/api/reset

// 3. Take actions
curl -X POST http://localhost:3000/api/step \
  -H "Content-Type: application/json" \
  -d '{"action": 1}'

// 4. Check state
curl http://localhost:3000/api/state

Use Cases

  • Training reinforcement learning agents (Q-Learning, DQN, etc.)
  • Testing pathfinding algorithms (A*, Dijkstra)
  • Educational purposes for AI/ML courses
  • Prototyping RL algorithms before scaling to complex environments
  • Comparing different RL approaches on a standard benchmark
